# Arkhé

Trabalho de conclusão do curso de desenvolvimento de sistemas da ETEC de Praia Grande.

## Instruções para rodar

Siga os passos para rodar o programa localmente:

1.  
    ```bash
    docker-compose build
    ```

2. 
    ```bash
    docker-compose up -d
    ```

O programa vai rodar no [http://localhost:8080](http://localhost:8080).
